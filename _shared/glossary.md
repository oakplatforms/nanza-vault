# Glossary — Shared Domain Vocabulary

Cross-project terms so everyone (and Claude) uses the same language. Add terms as they come up.

- **Listing** — a single item for sale.
- **Bulk / Lot** — a `BulkListing`: a listings-only grouping; appears as a distinct feed type. Child listings have visibility precedence rules.
- **Reference code** — short share code identifying an entity for deep links. Types include L (listing), G (group), U (profile).
- **Entity / Product** — catalog card data (an `Entity` has one `Product`).
- **Set** — a card set/release a brand ships.
- **Collection / List** — a user's saved grouping of items (`List` model, `SavedItem` entries).
- **Group** — a community/space; has membership and its own share code.
- **Caps / Credits** — the monetization model: lifetime-creation caps (listings, bids, messages, collections), enforced backend-side. Not a subscription.
